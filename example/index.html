      Certificate

      In this course, you will learn how to use more advanced features of Webflow - CMS, build the Style Guide that might later become full-fledged Design System, connect Webflow to the no-code software through Integrations and more. We’re going to learn on practical example of the Blog, that we’ll build from scratch and equip with some CMS superpowers. We’ll also allow our clients to edit readymade website and add articles, add powerful search engine and connect our own domain. Ready to get started?
